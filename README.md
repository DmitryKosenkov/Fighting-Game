# 2D Fighter Game

## Demo
You can play the game online here:  
[GitHub Pages link goes here](https://dmitrykosenkov.github.io/Fighting-Game/)

A simple 2D fighting game made with HTML, CSS, and JavaScript.  
Control two characters in a battle arena and try to defeat your opponent!

---

## Gameplay

- Two-player game with health bars and a timer.
- Each player can move, jump, and attack.
- The game ends when the timer runs out or one player's health reaches zero.

---

## Controls

### Player 1
- **Move Left:** `A`  
- **Move Right:** `D`  
- **Jump:** `W`  
- **Attack:** `Space`

### Player 2
- **Move Left:** `Arrow Left`  
- **Move Right:** `Arrow Right`  
- **Jump:** `Arrow Up`  
- **Attack:** `Arrow Down`
